# GitHub'a Yükleme Talimatları

Bu proje şu anda yerel git repository'sinde tutuluyor. GitHub'a yüklemek için:

## Adımlar:

1. GitHub'da yeni bir repository oluşturun
   - https://github.com/new adresine gidin
   - Repository adı: `rechnungsverwaltung-app` (veya istediğiniz ad)
   - Public veya Private seçin
   - README, .gitignore eklemeyin (zaten var)

2. Repository URL'sini alın
   - Oluşturduktan sonra göreceğiniz URL: `https://github.com/KULLANICI_ADI/repo-adi.git`

3. Aşağıdaki bilgileri bana verin:
   - GitHub repository URL'si
   - GitHub kullanıcı adı
   - GitHub Personal Access Token (Settings → Developer settings → Personal access tokens → Generate new token)
